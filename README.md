# Restart Explorer
A simple batch script that restarts Windows Explorer (explorer.exe) with one click.

What it does:

· Kills the explorer.exe process
· Restarts Explorer automatically
· Desktop and taskbar reload instantly

Why use it:

· Fixes frozen taskbar or desktop
· Refreshes icons after registry changes
· Quicker than rebooting
· No admin rights needed

How to use:

· Double-click Restart Explorer.bat
· Done.
